# Problem of the Day Solutions

This repository contains my solutions to the "Problem of the Day" (POTD) challenges from the GeeksforGeeks platform. Each day, GeeksforGeeks publishes a new coding problem for practice, and this repository houses my solutions to these problems.
## About the Problems

The problems in this repository are sourced from the GeeksforGeeks platform's daily POTD section. These challenges cover a wide range of topics including algorithms, data structures, dynamic programming, and more. Each problem comes with a detailed problem statement and requires a solution in a specific programming language.

## Language Used:

- Java

## Usage

Feel free to explore the solutions and use them for reference or learning purposes. Each problem has its own dedicated folder within the language-specific directories. You can navigate to the specific problem you are interested in and find the corresponding solution file. The code is well-documented and should be easy to understand.

To use the code in this repository, you can clone the repository to your local machine using the following command:

git clone https://github.com/Samyak072/GFG_POTD.git
